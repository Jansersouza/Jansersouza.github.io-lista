25. Crie uma página HTML e descreva, exemplificando, as funções das Tag’s abaixo: ALIGN VALIGN CELLSPACING CELLPADDING ROWSPAN COLSPAN WIDTH


Tag Align: Este atributo enumerado indica como a tabela deve ser alinhada considerando o conteúdo do documento. Que pode ter os seguintes valores: left, significando que a tabela deve ser exibida à esquerda do documento; center, significando que a tabela deve ser exibida centralizada no documento; right, significando que a tabela deve ser exibida à direita do documento.


Tag Valign: pode alinhar o conteudo das colunas verticalmente. Aceita os valores TOP, BOTTOM E CENTER.


Tag Cellspacing: define a distância da borda da célula até o seu conteúdo.


Tag Cellpadding: define a distância entre as células da tabela.


Tag Rowspan: usa-se para fazer uma céula ocupar o tamanho de várias linhas.


Tag Colspan: usa-se para fazer uma céula ocupar o tamanho de várias colunas.


Tag Width: usa-se para definir a largura da célula.
